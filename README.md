# Dokanki

Convert google docs, pdfs, html and other structured documents to Anki flashcards.

## Supported formats
- Google Doc
- PDF
- HTML
- DOCX

## Usage
```cmd
./dokanki.py 
    -name NAME
    -id ID
    -level 2
    -repetition 10,20,30
    -out file.apkg
    -type [inferred]
    // URL
```
